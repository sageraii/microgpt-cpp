# Ch00: 환경 설정

## 학습 목표

- Python이 설치되어 있는지 확인한다
- 화면에 글자를 출력하는 첫 번째 프로그램을 실행한다
- 이 튜토리얼에서 사용할 세 가지 모듈을 파악한다
- 우리가 만들 것이 무엇인지 큰 그림을 그린다

---

## 개념 설명

### Python이란?

Python은 사람이 읽기 쉬운 프로그래밍 언어입니다. 컴퓨터에게 할 일을 한국어처럼 자연스럽게 적어주면, 컴퓨터가 그 지시를 따라 실행합니다.

```
[여러분이 쓰는 코드] → Python → [컴퓨터가 실행]
     print("안녕")   →  통역사  →  화면에 "안녕" 출력
```

### 터미널이란?

터미널(Terminal)은 컴퓨터에게 타이핑으로 명령을 내리는 창입니다. 마우스 대신 키보드로 대화하는 느낌입니다.

```
터미널 창
┌─────────────────────────────────────┐
│ $ python --version                  │  ← 여러분이 입력
│ Python 3.11.0                       │  ← 컴퓨터의 답변
│ $                                   │  ← 다음 명령 기다림
└─────────────────────────────────────┘
```

## 코드 작성

### 1단계: Python 설치 확인

터미널을 열고 다음을 입력합니다.

```bash
python --version
```

**실행 결과:**
```
Python 3.11.0
```

숫자가 `3.8` 이상이면 됩니다. 만약 `command not found` 오류가 나오면 `python3 --version`을 시도해 보세요.

---

### 2단계: 첫 번째 프로그램 — Hello World

새 파일을 만들어 봅시다. `hello.py`라는 이름으로 저장합니다.

```python
# hello.py

print("hello, GPT!")   # 화면에 글자를 출력하는 함수
```

- `#` 기호 뒤의 글자는 **주석(comment)** 입니다. 컴퓨터는 이 부분을 무시합니다. 사람을 위한 설명입니다.
- `print(...)` 는 괄호 안의 내용을 화면에 출력하는 함수입니다.
- `"hello, GPT!"` 의 큰따옴표는 "이 안의 내용은 글자입니다"라는 표시입니다.

**실행:**
```bash
python hello.py
```

**실행 결과:**
```
hello, GPT!
```

---

### 3단계: 필요한 모듈 소개

모듈(module)은 미리 만들어진 도구 상자입니다. Python에는 수천 개의 모듈이 있는데, 이 튜토리얼에서는 딱 세 가지만 씁니다. 그리고 **추가 설치가 전혀 필요 없습니다!** Python을 설치하면 이 세 가지는 이미 들어 있습니다.

```python
# modules_intro.py

import os       # 파일이 있는지 확인하는 도구
import math     # 수학 함수 (log, exp 등)
import random   # 무작위 숫자와 섞기 도구

# os 사용 예시: 파일이 존재하는지 확인
print(os.path.exists("hello.py"))   # True 또는 False 출력

# math 사용 예시: 자연로그
print(math.log(10))                 # 2.302585...

# random 사용 예시: 무작위로 섞기
names = ["alice", "bob", "charlie"]
random.shuffle(names)
print(names)                        # 순서가 섞인 리스트
```

- `import 모듈이름` — 도구 상자를 꺼내 오는 명령입니다.
- `os.path.exists(경로)` — 해당 경로에 파일이 있으면 `True`, 없으면 `False`를 돌려줍니다.
- `math.log(숫자)` — 자연로그 값을 계산합니다.
- `random.shuffle(리스트)` — 리스트의 원소 순서를 뒤섞습니다.

**실행 결과 예시:**
```
True
2.302585092994046
['charlie', 'alice', 'bob']
```

## 이 튜토리얼에서 만들 것

우리가 만들 것은 **이름을 학습하고 새 이름을 지어내는 작은 GPT**입니다.

```
[학습 단계]
  "emma", "olivia", "ava", ... 32,000개의 이름을 읽는다
                    ↓
  신경망이 "이름다운 글자 순서"를 학습한다
                    ↓
[생성 단계]
  학습된 신경망이 새로운 이름을 만들어 낸다
  예: "alysa", "maren", "jorin", ...
```

ChatGPT나 GPT-4가 이 원리를 수억 배 크게 키운 것입니다. 기본 원리는 동일합니다.

가장 놀라운 사실은, **이 모든 것이 단 200줄의 Python 코드로 구현**된다는 점입니다. 외부 라이브러리(TensorFlow, PyTorch 등)를 전혀 쓰지 않습니다. `os`, `math`, `random` 세 가지만으로 GPT를 처음부터 만듭니다.

전체 코드 미리 보기:

```
microgpt.py (약 200줄)
├── 1~20줄   : 데이터셋 로드
├── 21~30줄  : 토크나이저
├── 31~73줄  : 자동 미분 (Autograd)
├── 74~90줄  : 모델 가중치 초기화
├── 91~144줄 : GPT 모델 구조 정의
├── 145~185줄: 학습 루프 (Adam 옵티마이저)
└── 186~201줄: 추론 (이름 생성)
```

이 튜토리얼은 각 부분을 한 줄씩 이해하며 직접 써 보는 것이 목표입니다.

## 핵심 정리

| 개념 | 설명 |
|------|------|
| `python --version` | 설치된 Python 버전 확인 |
